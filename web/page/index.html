关于我们

大中华区主机用户社区（Chinese System z User Community）是一个非盈利性技术组织，由同济大学、IBM硅谷实验室、国际DB2用户协会（IDUG）于2008年共同发起，提供国内主机领域最为资深与权威的用户交流平台，致力于主机开发与管理领域的技术共享、探讨、研究、合作及主机人才培养。
